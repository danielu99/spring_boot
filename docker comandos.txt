Comando para generar archivo jar de config-server: .\mvnw clean package
Comando para crear imagen de config-server: docker build -t config-server:v1 .
Comando para crear red para las imágenes de docker: docker network create springcloud
Comando para ejecutar la imagen para crear contenedor: docker run -p 8887:8887 --name config-server --network springcloud config-server:v1 // El interno es el de la derecha, ese debe ser el mismo que el del micro.
Comando para ver los contenedores (Solo LEVANTADOS): docker container ls
Comando para ver las imagenes de docker: docker images
Comando para ver todos los contenedores (Incluso los que se han cerrado): docker ps -a
Comando para descargar imagen de mysql: docker pull mysql:8
Comando para echar a andar imagen en un contenedor, mysql: docker run -p 3307:3306 --name microservicios-mysql8 --network springcloud -e MYSQL_ROOT_PASSWORD=root -e MYSQL_DATABASE=db_springboot_cloud -d mysql:8
Comando para ver logs: docker logs -f microservicios-mysql8
Comando para descargar imagen de postgresql: docker pull postgres:12-alpine
Comando para echar a andar imagen en un contenedor, postgresql: docker run -p 5433:5432 --name microservicios-postgres12 --network springcloud -e POSTGRES_PASSWORD=root -e POSTGRES_DB=db_springboot_cloud -d postgres:12-alpine
Comando para generar el jar BRINCANDO los tests, debido a que no encontrará las bases de datos: .\mvnw clean package -DskipTests

We added a VOLUME pointing to "/tmp" because that is where a Spring Boot application creates working directories for Tomcat by default. The effect is to create a temporary file on your host under "/var/lib/docker"
and link it to the container under "/tmp".

Comando para levantar contenedores a partir de un archivo docker-compose: docker-compose up
Comando para detener contenedores a través de docker-compose (elimina los contenedores y la red de springcloud): docker-compose down -v
Comando para levantar contenedor a partir de archivo docker-compose (-d lo hace en segundo plano y no muestra consola): docker-compose -d servicio-productos
